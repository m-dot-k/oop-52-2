Команды один раз на устройство

Команды для регистрации ваших данных
git config --global user.name "Your name"
git config --global user.email "Your email"
-------------------------------------------------------

Команды один раз для проекта
git init
git remote add origin https://github.com/m-dot-k/oop-52-2.git
---------------------------------------------------------------

Команды для частого использованияg

Команда для получения статуса проекта
git status

Команда для сохранения проекта в буфер
git add .

Команда для создания коммита или сохранения всего что есть
git commit -m "LS: lesson 1 added learn python oop"

Команда для отправки коммита в репозиторий
git push origin main

Команда для просмотра ветки
git branch

Команда для удаления директории и всего что она содержит
git rm -r --cached .idea (.idea - объект удаления)
